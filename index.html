Capabilities

Built for Enterprise Security Teams

CIS Benchmark Aligned

Scanners map to CIS Benchmarks for AWS, Azure, GCP, Windows, and Linux. Demonstrate compliance with industry-recognized baselines.

CI/CD Native

Every scanner exits with code 1 on CRITICAL/HIGH findings. Drop into GitHub Actions, GitLab CI, Jenkins, or any pipeline.

Multi-Format Reports

JSON for automation, interactive HTML for stakeholders, SARIF for GitHub Code Scanning, CSV for spreadsheets. All self-contained.

Read-Only by Design

All API scanners are strictly read-only. They never modify configurations, create resources, or alter your environment.

Zero Dependencies*

Most scanners use Python stdlib only. No bloated frameworks, no vendor SDKs, no license fees. *Network scanners need SSH/SNMP libs.

MITRE ATT&CK Mapped

Detection rules and CrowdStrike tools map findings to MITRE ATT&CK techniques. Speak the language your SOC already knows.

Architecture

Unified Scanner Architecture

Every scanner follows the same battle-tested 3-phase pipeline:

1

Collect

Connect to targets via SSH, SNMP, REST API, or parse files directly. Retry with exponential backoff. Never write.

2

Analyze

Evaluate collected data against security rules. Each finding gets an ID, severity, evidence, description, and remediation.

3

Report

Calculate posture scores, generate JSON + HTML reports, return appropriate exit codes for CI/CD pipelines.
